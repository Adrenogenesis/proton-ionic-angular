Réalisations Applications web Les projets les plus courant se déclinent en plusieurs catégories :
  • Les sites internet : Type vitrine, dynamique et e-commerce
  • Les CMS : Content Management System, c’est-à-dire en français « système de gestion de contenus », comme Wordpress
  • Les applications web : Questionnaires, comptabilité etc...
  • Création graphique : Logo, carte de visite, flyers.
  • Campagnes publicitaire : Mailing
  • Gestion des données : Base de données, MySql, ELK.
